---
title: "PnL Statistic"
api: "POST https://api.copin.io/public/position/statistic/pnl-statistics"
description: "Retrieve pnl position statistics"
---

## Authentication
This endpoint requires authentication using a JWT token in the Authorization header.

<ParamField header="Authorization" type="string">
    JWT token in the format: `Authorization {token}`
</ParamField>

<ParamField header="x-api-key" type="string">
    Token in the format: `x-api-key {api-key}`
</ParamField>

## Request Body

<ParamField body="accounts" type="array" required>
   Accounts list for filter
    <Expandable title="properties">
        <ParamField body="account" type="string">
            Account for filter
        </ParamField>
        <ParamField body="protocol" type="string">
            Protocol for filter (KILOEX_BASE, GMX, GNS...)
        </ParamField>
    </Expandable>
</ParamField>
<ParamField body="statisticType" type="string">
    Statistic for different time periods (FULL, D60, D30, D15, D7).
</ParamField>

## Response

The response is structured as an object where each key is an account address from your request. For each account, the API returns time series data containing dates and corresponding trading metrics.
<ResponseField name="{account_address}" type="object">
    Data for each requested account, keyed by account address
    <Expandable title="properties">
        <ResponseField name="date" type="array">
            Array of dates for the time series in ISO format
        </ResponseField>
        <ResponseField name="realisedPnl" type="array">
            Array of realised profit and loss values corresponding to each date
        </ResponseField>
        <ResponseField name="unrealisedPnl" type="array">
            Array of unrealised profit and loss values corresponding to each date
        </ResponseField>
        <ResponseField name="fee" type="array">
            Array of trading fee values corresponding to each date
        </ResponseField>
    </Expandable>
</ResponseField>
