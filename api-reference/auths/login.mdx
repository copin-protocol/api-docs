---
title: "Login"
api: "POST https://api.copin.io/auth/web3/login"
description: "Web3 authentication login endpoint to obtain verification code"
---

## Request Body

<ParamField body="address" type="string" required>
    The Ethereum wallet address of the user attempting to login
</ParamField>

## Request Example

```bash
curl --request POST \
--url "https://api.copin.io/auth/web3/login" \
--header 'Content-Type: application/json' \
--data '{"address":"0x1535484c1eec1d203fe1a53ea11a621a884ae067"}'
```
## Response
<ResponseField name="verifyCode" type="string">
  A verification code that will be used in subsequent authentication steps
</ResponseField>
