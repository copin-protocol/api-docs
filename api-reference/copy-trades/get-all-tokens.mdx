---
title: "Get all tokens"
api: "GET https://api.copin.io/pairs/tokens"
description: "Retrieve tokens of all protocols"
---
## Request Example

```bash
curl --request GET \
--url "https://api.copin.io/pairs/tokens" \
--header 'Content-Type: application/json' \
```
## Response
The API returns a comprehensive list of tokens organized by protocol. For each protocol, an array of token objects is provided containing symbol information and associated blockchain details.
<ResponseField name="data" type="object">
    Container for token data grouped by protocol
    <Expandable title="properties">
        <ResponseField name="{protocol_name}" type="array">
            Array of token objects for each protocol (e.g., "APOLLOX_BASE")
            <Expandable title="properties">
                <ResponseField name="symbol" type="string">
                    Trading symbol of the token (e.g., "ETH", "BTC")
                </ResponseField>
                <ResponseField name="indexTokens" type="array">
                    Array of contract addresses associated with this token on the protocol
                </ResponseField>
                <ResponseField name="isForex" type="boolean">
                    Indicates whether the token is a forex-based trading pair
                </ResponseField>
            </Expandable>
        </ResponseField>
    </Expandable>
</ResponseField>
